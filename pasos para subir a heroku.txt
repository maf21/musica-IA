Configurar variables de entorno:
1. python -m venv venv
2. venv\Scripts\activate
3. flask run(comprobar si corre correctamente)
4. set FLASK_ENV=development

Configurar heroku
1. heroku login
2. echo "web: gunicorn app:app" > Procfile
3. python -m pip install gunicorn
4. heroku create musica-ia

Abrir heroku
1. heroku open
